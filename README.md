# School 1298 Telegram Bot

Бот для авторизации пользователей школы 1298.

## Функциональность

- Авторизация по номеру телефона
- Подтверждение email
- Привязка MAX ID

## Установка

1. `npm install`
2. Создать `.env` файл с токеном бота
3. `npm start`

## Команды

- `/start` - начать авторизацию
- `/help` - показать справку
- `/status` - проверить статус
